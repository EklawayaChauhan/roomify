## 📖 Overview

Roomify is an AI-powered architectural visualization platform that converts 2D floor plans into realistic 3D renders. Users can upload floor plans, generate high-quality AI visualizations, manage projects, and share their work with others.

The application focuses on providing a modern, fast, and intuitive experience for architects, designers, students, and homeowners.

---

## ✨ Features

- 🏡 Convert 2D floor plans into AI-generated 3D visualizations
- 🤖 AI-powered rendering
- 📁 Project management dashboard
- 🖼️ Permanent image hosting
- 🌍 Public community gallery
- 🔒 Public & private project visibility
- 📊 Side-by-side comparison of original and generated images
- ⚡ Fast and responsive UI
- 📥 Export generated renders

## 🚀 Getting Started

### Prerequisites

- Node.js
- npm
- Git

### Clone the repository

```bash
git clone https://github.com/EklawayaChauhan/roomify.git

cd roomify
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the project root.

```env
VITE_PUTER_WORKER_URL=""
```

Replace the value with your own Puter Worker URL.

---

## ▶️ Run the Project

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
